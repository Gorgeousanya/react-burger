# React-Burger

## Используемые технологии
- React
- Redux
- React-Thunk
- React DnD
- Jest
- Cypress

## Скачать проект
1. склонировать репозиторий: `git clone`
2. выполнить установку пакетов: `npm i`
3. запустить проект: `npm start`
